Helper Class

Floats

Use these classes to float an element left or right. !important is included to avoid specificity issues.

HTML:
<div class="pull-left"></div>
<div class="pull-right"></div>

Text Alignment

All text should be aligned left by default.
If it's specific to component, then define other text alignment in component styles.
Otherwise, use the classes below to align text in unique scenario.
!important is included to avoid specificity issues.

Heading 3

Dignissimos, aut labore corporis laborum illo facere officia!

Heading 3

Dignissimos, aut labore corporis laborum illo facere officia!

HTML:
<div class="text-right">Dignissimos, aut labore corporis laborum illo facere officia!</div>
<div class="text-center">Dignissimos, aut labore corporis laborum illo facere officia!</div>

Clearfix

Use this class to clear floats on an element.

HTML:
<div class="clearfix"></div>

Background Colors

Use these classes to quickly add a background colour to your components.

{{item.colorName}}
.{{item.className}}

This is a sample div with a silver background.

HTML:
<div class="component-class bg-silver">
   <p>This is a sample div with silver background.</p>
</div>
