Typography

These are the default typography styles for DBS Ideal. Further customisation to these styles should be added on component level.

Headings

Heading 1

HTML:
<h1>Heading 1</h1>

SCSS:
h1 {{ '{' }}
    font-size: em(32);
    font-family: $font-family;
    font-weight: $font-weight-medium;
    line-height: (32/32);
{{ '}' }}

CSS:
h1 {{ '{' }}
    font-size: 32px;
    font-family: 'FrutigerNextPro', Arial, Helvetica, sans-serif;
    font-weight: 500;
    line-height: 32px;
{{ '}' }}
