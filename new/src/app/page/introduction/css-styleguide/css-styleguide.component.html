CSS/SCSS Coding Styleguide

We are using SCSS to manage and compile our CSS. This document serves as a guideline to keep formatting of SCSS files consistent. General rules include:

  1. Be consistant with indentation
  2. Be consistant about where spaces before/after colons/braces go
  3. One selector per line, One rule per line
  4. List related properties together
  5. Don't use ID's #idname
  6. Use logical class names

Formatting Guidelines

List @extend(s) First

.button-red {{ '{' }}
    @extend %button;
    ...
{{ '}' }}

List @include(s) Next

.button-red {{ '{' }}
    @extend %button;
    @include clearfix;
    ...
{{ '}' }}

List "Regular" Styles Next

.button-red {{ '{' }}
    @extend %button;
    @include clearfix;
    background: $dbsred;
    ...
{{ '}' }}

Nested Pseudo Classes and Pseudo Elements Next

.button-red {{ '{' }}
    @extend %button;
    @include clearfix;
    background: $dbsred;
    &:hover {{ '{' }}
        background: $dbsred;
    {{ '}' }}
    &::before {{ '{' }}
        content: "";
        display: block;
    {{ '}' }}
     ...
{{ '}' }}

Nested Selectors Last

.button-red {{ '{' }}
    @extend %button;
    @include clearfix;
    background: $dbsred;
    &:hover {{ '{' }}
        background: $dbsred;
    {{ '}' }}
    &::before {{ '{' }}
        content: "";
        display: block;
    {{ '}' }}
    > h3 {{ '{' }}
        @include transform(rotate(90deg));
        border-bottom: 1px solid white;
    {{ '}' }}
     ...
{{ '}' }}

Syntax Formatting

Lowercase Writing

This includes:

  • all selectors: .classname
  • hex codes: #fff;

Shorthand Values

Use shorthand values where possible

color: #fff;
instead of
color: #ffffff;

margin: 10px 5px;
instead of
color: 10px 5px 10px 5px;

Line Height

Define line-height in decimal without any units

For example, if the body font is 16px with 20px line-height, then the line-height should be defined as:

line-height: 1.25;
instead of
line-height: 20px;
